# Spaghetti

A satirical, deadpan-serious informational website about spaghetti. Features a timeline of spaghetti's "history", a taxonomy of spaghetti species, absurd recipes, and a quiz.

**This is a test/demo project for [FOUX](https://github.com/fouxproject).** It exists to provide a realistic web app for testing the FOUX Chrome DevTools extension, which enables visual design iteration on live sites. Generated in a single shot by Claude. This one is intentionally bad code.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS (primary) + CSS Modules + plain CSS + inline styles (intentionally mixed)
- React Router v6

**Code style**: intentionally inconsistent. Mixed file naming conventions, mixed export styles, mixed state management patterns (`useState`, `useReducer`), over-engineered and under-engineered components side by side. Simulates accumulated tech debt from a team with no style guide.

## Running

```
npm install
npm run dev
```
